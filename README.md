# Comment Classification ML Project

## Overview
This project predicts comment categories using NLP and machine learning models.

## Models Used
- Logistic Regression
- SVM
- LightGBM

## Techniques
- TF-IDF vectorization
- OneHot Encoding
- Standard Scaling

## Evaluation Metric
Macro F1 Score

## Results
Best model: LightGBM (~0.81 macro F1)

## Dataset
Kaggle Competition Dataset
